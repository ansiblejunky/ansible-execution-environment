## Installation

Run the following steps using `pyenv`.

```shell
pyenv install 3.8.9
pyenv activate 3.8.9 ansible-ee
pip install ansible-navigator
pip install ansible-builder
```

## Build

The first step is to build your execution environment, which is just building an image.
